Welcome to the Font Styling Example

This is an example of a simple web page demonstrating the use of different fonts. The main font for the body text is Roboto, which is a modern sans-serif font commonly used for clean and readable design.

The heading uses the Lobster font, which has a cursive style and is often used for titles or decorative text.

Additionally, we can use classic web-safe fonts like Courier New for a monospace style, which is great for displaying code or data.
